# Subtract from Hero-Range 

Use [<code>not</code>](/rangenot)  command to subtract range. 

<div class="tip custom-block">
<p>

If your <code>hero-range</code>  already contains some range, command <code>top0</code> will clear it.
 
</p>
</div>
